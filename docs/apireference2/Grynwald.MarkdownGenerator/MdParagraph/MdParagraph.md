## Remarks

The content spans will be wrapped in a instance of [`MdCompositeSpan`](../MdCompositeSpan.md). Thus `new MdParagraph(span1, span2)` is equivalent to `new MdParagraph(new MdCompositeSpan(span1, span2))`
