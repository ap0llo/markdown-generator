# MdSerializationOptions class

Encapsulates settings that control how a document is serialized. A instance of [`MdSerializationOptions`](./MdSerializationOptions.md) can be passed to the ToString overloads in [`MdBlock`](./MdBlock.md), [`MdSpan`](./MdSpan.md) and [`MdDocument`](./MdDocument.md) as well as the Save() method of [`MdDocument`](./MdDocument.md)
