# EnumerableExtensions.Join method (1 of 3)

Combines the sequence of spans to a single span.

```csharp
public static MdSpan Join(this IEnumerable<MdSpan> spans)
```

| parameter | description |
| --- | --- |
| spans | The sequence of spans to combine |

## Return Value

Returns a new span containing all the specified spans.

* If *spans* is empty, an instance of [`MdEmptySpan`](../MdEmptySpan.md) will be returned
* If *spans* contains a single item, this single span will be returned
* Otherwise a instance of [`MdCompositeSpan`](../MdCompositeSpan.md) is returned that wraps all the specified spans.
