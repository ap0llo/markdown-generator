# MdSerializationOptions.TextFormatter property

Gets or sets the implementation to use for escaping text when saving a Markdown document.

```csharp
public ITextFormatter? TextFormatter { get; set; }
```

## Remarks

The implementation of [`ITextFormatter`](../ITextFormatter.md) set here will be used to escape plain text before writing it to the output document.

When no escaper is set, [`DefaultTextFormatter`](../DefaultTextFormatter.md) will be used.
