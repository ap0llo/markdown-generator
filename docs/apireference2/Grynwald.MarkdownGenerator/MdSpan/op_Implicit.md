# MdSpan Implicit operator

Implicitly creates a [`MdSpan`](../MdSpan.md) from a string. The string value will be wrapped into an instance of [`MdTextSpan`](../MdTextSpan.md) and thus be escaped in the output.

```csharp
public static implicit operator MdSpan(string? text)
```

| parameter | description |
| --- | --- |
| text | The string value to wrap in a span. |

## Remarks

Wraps a string in an instance of [`MdTextSpan`](../MdTextSpan.md). Although the nullable annotations do not indicate it, passing in `null` will return `null`.
