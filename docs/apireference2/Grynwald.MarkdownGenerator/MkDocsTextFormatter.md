## Remarks

This implementation works largely the same as [`DefaultTextFormatter`](./DefaultTextFormatter.md) but changes how some characters are escaped:

* `<` is escaped as `&lt;`.
* `>` is escaped as `&gt;`.
* `/` is not escaped.
