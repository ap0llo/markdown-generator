## Remarks

[`DocumentSet`](./DocumentSet-1.md) allows grouping of multiple [`IDocument`](./IDocument.md) instances and associating them with a path. The document set offers methods to create links between documents and saving all documents to a directory at once.

It implements a 1:1 mapping between documents and paths.
