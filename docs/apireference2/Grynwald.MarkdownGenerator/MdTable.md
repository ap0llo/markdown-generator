## Remarks

CommonMark does not specify a table format, but tables are common in GitHub Flavored Markdown. Tables can be serialized either as GitHub Flavored Markdown tables (default) or inline HTML-tables. For details see [`TableStyle`](./MdSerializationOptions/TableStyle.md)
