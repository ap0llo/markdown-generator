## Remarks

[`MdDocumentSet`](./MdDocumentSet.md) allows grouping of multiple [`MdDocument`](./MdDocument.md) instances and associating them with a path. The document set offers methods to create links between documents and saving all documents to a directory at once.

[`MdDocumentSet`](./MdDocumentSet.md) implements a 1:1 mapping between documents and paths.
